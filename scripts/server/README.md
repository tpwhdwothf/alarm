# 서버용 스크립트

이 폴더의 스크립트는 **Oracle Cloud 실서버**에서 실행합니다.

## start-alarm.sh

재부팅 시 **봇만** 자동 시작하고, 시세 워커(worker-kis)는 시작하지 않습니다.

- **역할**: `@reboot` cron에서 호출 시 봇만 켜고, worker-kis는 중지 후 저장
- **사용 시점**: 시세 조회를 끄고 봇만 24시간 돌릴 때
- **설치 방법**: `docs/실서버_배포_가이드.md` → "시세 조회(워커)만 끄기" → "2) 재부팅 시에도 워커가 자동으로 켜지지 않게 하기"

## ssh-debug-logger.sh

SSH 끊김 원인 확인용 로그 수집 스크립트.

- **역할**: 2~3분마다 메모리 사용률, 부하, SSH 연결 수를 `~/ssh-debug.log`에 기록
- **사용 시점**: SSH가 자주 끊길 때, 끊김 직전 시스템 상태를 확인하기 위함
- **설치 방법**: `docs/실서버_배포_가이드.md` → "SSH 끊김 원인 확인" 섹션 참고
